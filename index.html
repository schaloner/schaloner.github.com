Resolvers

To add resolvers for this repository to your Play 2 project, use

val main = PlayProject(appName, appVersion, appDependencies, mainLang = JAVA).settings(
  resolvers += Resolver.url("Objectify Play Repository", url("http://schaloner.github.io/releases/"))(Resolver.ivyStylePatterns),
  resolvers += Resolver.url("Objectify Play Snapshot Repository", url("http://schaloner.github.io/snapshots/"))(Resolver.ivyStylePatterns)
)

Don't forget to change the mainLang as necessary!

Deadbolt 2

Play 2.0

To use the latest release version of Deadbolt 2 in your project, use

val appDependencies = Seq(
  "be.objectify" %% "deadbolt-2" % "1.1.3"
)

Play 2.1

For Play 2.1, you need to use the Deadbolt 2.1 in your project. Add one or both of the following according to your project needs

val appDependencies = Seq(
  "be.objectify" %% "deadbolt-java" % "2.1.2",
  "be.objectify" %% "deadbolt-scala" % "2.1"
)

Play 2.2

For Play 2.2, you need to use the Deadbolt 2.2 in your project. Add one or both of the following according to your project needs

val appDependencies = Seq(
  "be.objectify" %% "deadbolt-java" % "2.2-RC4",
  "be.objectify" %% "deadbolt-scala" % "2.2-RC2"
)

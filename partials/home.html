Deadbolt

Deadbolt is a powerful, comprehensive authorization module for the Play framework.

Java and Scala APIs

Deadbolt has idiomatic APIs for Java and Scala.

Controller- and view-level auth

An extensible range of static and dynamic authorisations at the controller level. Similarly, you can customise your UI based on a user's authorization levels.

Plays well with others

Authentication support is pluggable - use your favourite tool to let users log in.

Repository

Deadbolt is currently hosted in the Objectify repository. At some point in the future, it will move to Maven Central. In the meantime, you need to add the following to your Play project: 
resolvers += Resolver.url("Objectify Play Repository", url("http://schaloner.github.io/releases/"))(Resolver.ivyStylePatterns),
resolvers += Resolver.url("Objectify Play Snapshot Repository", url("http://schaloner.github.io/snapshots/"))(Resolver.ivyStylePatterns)
